Overview

This project involved designing the complete user interface for a next-generation smart home ecosystem. The core problem was the fragmentation and clunky UX of existing IoT apps. This design system provides a centralized, ultra-intuitive interface allowing users to control lighting, climate, and security with minimal friction and engaging micro-interactions.

[!CAUTION] The Challenge Designing an interface that is simple enough for non-technical users but powerful enough for automation power-users. Creating a cohesive visual language that translates well across both mobile screens and larger wall-mounted tablet displays.

[!TIP] The Solution Developed a comprehensive component library in Figma utilizing auto-layout and strict design tokens. I introduced a 'progressive disclosure' pattern where advanced automation settings are tucked behind intuitive gestures. Advanced prototyping with Framer Motion was used to validate micro-interactions and the spatial model of the interface before any code was written.

Technical Challenges

The Problem

Creating a spatial model that felt intuitive across both mobile and tablet orientations.

The Solution

Developed a modular card-based system with CSS Container Queries for truly context-aware layouts.

The Problem

Conveying complex device states (e.g., partial heating) through simple icons.

The Solution

Iterated through 3 rounds of user testing to refine a bespoke animated SVG icon set.

Project Outcomes

Reduced task completion time for setup procedures by 45% in usability tests.

Design system successfully adopted for both iOS and Android prototypes.

High fidelity prototypes praised for 'premium physical feel' of the UI.
