## Overview
The very first iteration of my personal portfolio. Designed to be deliberately raw, it adopted brutalist design principles to stand out in a sea of standard corporate templates. It solved the problem of establishing an initial web presence while serving as an experimental playground for vanilla JavaScript and CSS grid layouts.

> [!CAUTION] The Challenge
> Achieving complex, responsive grid layouts utilizing pure CSS without the aid of modern frameworks. Writing lightweight vanilla JavaScript to handle smooth scroll behaviors and simple animations while ensuring absolute maximum performance and zero dependencies.

> [!TIP] The Solution
> Relied heavily on CSS Variables (Custom Properties) and CSS Grid to construct a rigid but fully responsive typographic layout. I wrote custom, minimalistic JavaScript observer classes to handle lazy-loading for images and trigger threshold-based scroll animations, resulting in a perfect 100/100 Lighthouse score.
